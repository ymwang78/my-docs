# Project窗口实现总结

## 概述
已成功实现了一个功能完整的可停靠项目管理窗口，参考xRto的tool widget设计，提供了项目文件的层级管理和运行实例监控功能。

## 实现的功能

### 1. 停靠窗口特性
- **位置**: 默认停靠在主窗口左侧
- **可见性**: 默认隐藏，可通过菜单"视图(V) → 项目管理(P)"显示/隐藏
- **停靠区域**: 支持左侧和右侧停靠
- **特性**: 可移动、可浮动、可关闭
- **大小**: 默认300x600像素，可调整

### 2. 项目文件管理
- **支持格式**: JSON (.json) 和 YAML (.yaml, .yml) 项目文件
- **拖拽添加**: 支持将项目文件拖拽到窗口中添加
- **文件验证**: 自动验证文件格式和内容有效性
- **层级显示**: 项目文件作为父节点，运行实例作为子节点

### 3. 运行实例管理
- **实例类型**: 
  - In-Process (同进程运行)
  - Local Backend (本地后台)
  - Remote Instance (远程实例)
- **状态显示**: 运行状态用颜色区分（绿色=运行中，红色=已停止）
- **实例操作**: 启动、停止、在线调优、实时曲线等

### 4. 用户交互
- **右键菜单**: 根据节点类型显示不同的上下文菜单
- **单击选择**: 选择项目文件或运行实例
- **双击操作**: 双击项目文件打开配置编辑器，双击实例打开监控仪表盘
- **键盘快捷键**: Delete删除项目，F5刷新列表

### 5. 搜索和过滤
- **搜索框**: 实时搜索项目名称
- **视图模式**: 支持树形、列表等不同显示模式
- **过滤选项**: 按项目类型、状态等过滤

### 6. 工具栏功能
- **新建项目**: 创建新的项目文件
- **打开项目**: 打开现有项目文件
- **保存项目**: 保存当前项目
- **刷新**: 刷新项目列表
- **折叠/展开**: 控制树形结构的展开状态

## 技术实现

### 核心类
1. **ProjectDockWidget**: 停靠窗口容器
   - 继承自QDockWidget
   - 管理工具栏、搜索栏、状态栏
   - 处理窗口的停靠和浮动

2. **ProjectTreeWidget**: 项目树控件
   - 继承自QTreeWidget
   - 实现拖拽功能
   - 管理项目文件和运行实例的层级结构
   - 处理右键菜单和用户交互

### 信号和槽
- **项目选择**: `projectFileSelected`, `projectFileDoubleClicked`
- **实例选择**: `instanceSelected`, `instanceDoubleClicked`
- **项目操作**: `projectFileOpened`, `projectFileRemoved`, `projectFileCreated`
- **实例操作**: `instanceStarted`, `instanceStopped`

### 数据管理
- **ProjectInfo结构**: 存储项目文件信息和运行实例列表
- **QSettings**: 保存窗口几何形状和项目文件列表
- **文件验证**: JSON解析验证和YAML格式检查

## 集成到主窗口

### MainWindow集成
- 在`setupDockWidgets()`中创建ProjectDockWidget
- 在`setupConnections()`中连接信号和槽
- 在菜单栏添加"项目管理"选项
- 实现`onViewProjectManager()`控制窗口显示/隐藏

### 视图切换
- 选择项目文件时切换到配置编辑器视图
- 选择运行实例时切换到实时监控仪表盘
- 状态栏显示相应的状态信息

## 使用说明

### 显示Project窗口
1. 启动xApc程序
2. 点击菜单"视图(V) → 项目管理(P)"
3. Project窗口将出现在主窗口左侧

### 添加项目文件
1. 将JSON或YAML项目文件拖拽到Project窗口中
2. 文件将作为项目节点显示在树中
3. 项目节点下会自动显示运行实例

### 管理运行实例
1. 右键点击项目节点，选择相应的操作
2. 右键点击实例节点，可以启动、停止或监控实例
3. 双击实例节点打开实时监控仪表盘
