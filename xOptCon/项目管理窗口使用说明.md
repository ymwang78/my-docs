# 项目管理窗口使用说明

## 概述

项目管理窗口是一个可停靠的窗口，提供了以项目为核心的层级视图，用于管理MPC控制器的项目文件和运行实例。

## 功能特性

### 1. 项目层级结构

项目管理窗口采用以下层级结构：

```
▼ 项目 (Projects)
  ├─ ▼ 📄 Boiler_MPC.json
  │    ├─ ▶️ 在GUI中运行 (In-Process)
  │    └─ ▶️ 远程实例 (192.168.1.100:8080)
  │
  ├─ ▼ 📄 Furnace_Control.yaml
  │    └─ ▶️ 本机后台 (localhost:5556)
  │
  └─ 📄 Distillation_Column.json (未运行)
```

### 2. 项目文件节点

- **图标**: 📄 表示项目配置文件
- **功能**: 代表项目的"设计蓝图"和"控制中心"
- **点击行为**: 点击项目文件节点会切换到"控制器配置编辑器"
- **右键菜单**: 提供新建、打开、保存、运行、部署等操作

### 3. 运行实例节点

- **图标**: ▶️ 表示正在运行的控制器实例
- **功能**: 代表活动的、正在运行的控制器实例
- **点击行为**: 点击运行实例节点会切换到"实时监控仪表盘"
- **右键菜单**: 提供停止实例等操作

## 使用方法

### 1. 显示/隐藏项目管理窗口

- **菜单方式**: 视图 → 项目管理
- **快捷键**: 无（可自定义）
- **默认状态**: 隐藏，停靠在左侧

### 2. 工具栏功能

项目管理窗口顶部提供以下工具栏按钮：

- **新建项目** (Ctrl+N): 创建新的MPC项目
- **打开项目** (Ctrl+O): 打开现有的项目文件
- **保存项目** (Ctrl+S): 保存当前项目
- **刷新** (F5): 刷新项目列表
- **全部折叠**: 折叠所有项目节点
- **全部展开**: 展开所有项目节点

### 3. 搜索和过滤

- **搜索框**: 输入关键词搜索项目名称
- **视图模式**: 选择树视图、列表视图或图标视图
- **过滤器**: 按运行状态或文件类型过滤项目

### 4. 项目操作

#### 4.1 创建新项目

1. 点击工具栏的"新建项目"按钮
2. 选择保存位置和文件名
3. 选择项目模板（可选）
4. 项目文件创建后会自动添加到项目列表

#### 4.2 打开项目

1. 点击工具栏的"打开项目"按钮
2. 选择项目文件（支持.json、.yaml、.yml格式）
3. 项目文件会添加到项目列表并显示在树中

#### 4.3 运行项目

右键点击项目文件节点，选择以下运行方式：

- **在GUI中运行**: 在当前应用程序进程中运行控制器
- **部署到本机后台**: 在本地后台服务中运行控制器
- **部署到远程服务器**: 在远程服务器上运行控制器

#### 4.4 监控实例

- **单击实例**: 切换到实时监控仪表盘
- **双击实例**: 打开详细的监控视图
- **右键实例**: 停止运行中的实例

### 5. 拖放支持

- **拖入项目文件**: 将项目文件拖放到项目管理窗口来自动添加
- **支持格式**: .json、.yaml、.yml文件

### 6. 键盘快捷键

- **Delete**: 删除选中的项目（从列表中移除，不删除文件）
- **F5**: 刷新项目列表
- **Ctrl+N**: 新建项目
- **Ctrl+O**: 打开项目
- **Ctrl+S**: 保存项目

## 项目文件格式

### JSON格式示例

```json
{
  "name": "Boiler_MPC",
  "version": "1.0.0",
  "created": "2024-01-15T10:30:00Z",
  "template": "MPC_Controller",
  "description": "锅炉模型预测控制项目",
  "configuration": {
    "general": {
      "sampling_time": 1,
      "time_unit": "seconds"
    }
  },
  "variables": {
    "manipulated_variables": [...],
    "controlled_variables": [...],
    "disturbance_variables": [...]
  }
}
```

### YAML格式示例

```yaml
name: Furnace_Control
version: "1.0.0"
created: "2024-01-15T11:00:00Z"
template: "MPC_Controller"
description: "炉膛温度控制项目"
configuration:
  general:
    sampling_time: 2
    time_unit: "seconds"
variables:
  manipulated_variables:
    - name: "Gas_Valve"
      description: "燃气阀门开度"
      unit: "%"
```

## 状态显示

项目管理窗口底部显示当前状态信息：

- **就绪**: 正常状态
- **搜索: X 个项目**: 正在搜索，显示匹配的项目数量
- **过滤: X 个项目**: 正在过滤，显示符合条件的项目数量
- **X 个项目, Y 个运行实例**: 显示总项目数和运行实例数

## 设置和配置

项目管理窗口会自动保存以下设置：

- **窗口位置和大小**: 停靠位置、浮动状态、几何信息
- **项目列表**: 最近打开的项目文件
- **视图状态**: 展开/折叠状态

## 故障排除

### 常见问题

1. **项目文件无法加载**
   - 检查文件格式是否正确（JSON或YAML）
   - 确认文件路径是否有效
   - 查看错误日志获取详细信息

2. **运行实例无法启动**
   - 检查项目配置是否正确
   - 确认网络连接（远程部署时）
   - 查看系统日志

3. **项目管理窗口不显示**
   - 检查菜单"视图 → 项目管理"是否勾选
   - 确认窗口是否被其他窗口遮挡
   - 尝试重置窗口布局

### 日志查看

项目管理相关的日志信息会显示在主窗口的日志面板中，包括：

- 项目文件加载/保存状态
- 运行实例启动/停止状态
- 错误和警告信息
