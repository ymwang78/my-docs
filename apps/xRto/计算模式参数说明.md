# 计算模式参数说明

## 概述
xRto程序的计算模式支持命令行参数，允许无GUI模式下执行计算任务和脚本。

## 命令行参数

### 必需参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `--computeTask` | 指定要运行的计算任务名称 | `--computeTask task1` |
| `--projectFile` | 指定要加载的工程文件路径 | `--projectFile project.xrto` |

### 可选参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `--scriptFile` | 指定要执行的脚本文件路径（Python或Lua） | `--scriptFile script.py` |
| `--scriptArg` | 传递给脚本的参数字符串 | `--scriptArg "param1=value1"` |
| `--timeOut` | 指定超时秒数，超时后程序自动退出 | `--timeOut 300` |

## 使用示例

### 示例1：基本计算任务
```bash
xRto.exe --computeTask myTask --projectFile C:\Projects\project.xrto
```

### 示例2：执行脚本文件
```bash
xRto.exe --computeTask myTask --projectFile C:\Projects\project.xrto --scriptFile C:\Scripts\process.py
```

### 示例3：执行脚本并传递参数
```bash
xRto.exe --computeTask myTask --projectFile C:\Projects\project.xrto --scriptFile C:\Scripts\process.py --scriptArg "mode=production,output=result.csv"
```

### 示例4：设置超时时间
```bash
xRto.exe --computeTask myTask --projectFile C:\Projects\project.xrto --timeOut 600
```
说明：程序将在600秒（10分钟）后自动退出

### 示例5：完整示例（所有参数）
```bash
xRto.exe --computeTask optimization --projectFile C:\Projects\plant.xrto --scriptFile C:\Scripts\optimize.py --scriptArg "iterations=100,tolerance=0.001" --timeOut 1800
```

## 脚本中获取参数

### Python脚本
在Python脚本中，可以通过环境变量获取scriptArg参数：

```python
import os

# 获取传入的脚本参数
script_arg = os.environ.get('XRTO_SCRIPT_ARG', '')
print(f"接收到的参数: {script_arg}")

# 解析参数（如果使用key=value格式）
params = {}
if script_arg:
    for item in script_arg.split(','):
        if '=' in item:
            key, value = item.split('=', 1)
            params[key.strip()] = value.strip()

print(f"解析后的参数: {params}")

# 使用参数
iterations = int(params.get('iterations', '10'))
tolerance = float(params.get('tolerance', '0.01'))
```

### Lua脚本
在Lua脚本中，也可以通过环境变量获取scriptArg参数：

```lua
-- 获取传入的脚本参数
local script_arg = os.getenv('XRTO_SCRIPT_ARG') or ''
print("接收到的参数: " .. script_arg)

-- 解析参数
local params = {}
for item in string.gmatch(script_arg, "[^,]+") do
    local key, value = string.match(item, "([^=]+)=([^=]+)")
    if key and value then
        params[key:match("^%s*(.-)%s*$")] = value:match("^%s*(.-)%s*$")
    end
end

-- 使用参数
local iterations = tonumber(params.iterations) or 10
local tolerance = tonumber(params.tolerance) or 0.01
```

## 退出码说明

| 退出码 | 说明 |
|--------|------|
| `0` | 正常退出，任务执行成功 |
| `-1` | 错误退出，任务执行失败 |
| `-2` | 超时退出，程序运行超过指定时间 |
| `1` | 参数验证失败 |

## 注意事项

1. **参数顺序**：参数顺序不重要，但每个参数的值必须紧跟在参数名后面
2. **路径格式**：Windows路径可以使用反斜杠(`\`)或正斜杠(`/`)
3. **scriptArg格式**：scriptArg可以是任意字符串，建议使用key=value格式并用逗号分隔
4. **超时机制**：超时计时从程序启动开始，不是从任务开始执行时开始
5. **环境变量**：脚本参数通过环境变量`XRTO_SCRIPT_ARG`传递给脚本

## Python API使用

### 函数签名

```python
controller.runComputeTask(
    computeTaskName,      # 必需：计算任务名称
    scriptFile="",        # 可选：脚本文件路径
    timeOutSeconds=0,     # 可选：超时秒数（0表示无超时）
    scriptArg=""          # 可选：传递给脚本的参数字符串
)
```

### Python API示例

#### 基本用法
```python
# 获取控制器实例
controller = getFlowSheetController()

# 运行基本计算任务
process_id = controller.runComputeTask("optimization_task")
```

#### 使用脚本
```python
process_id = controller.runComputeTask(
    "optimization_task",
    scriptFile="process.py"
)
```

#### 设置超时
```python
process_id = controller.runComputeTask(
    "optimization_task",
    scriptFile="process.py",
    timeOutSeconds=600  # 10分钟超时
)
```

#### 传递参数
```python
process_id = controller.runComputeTask(
    "optimization_task",
    scriptFile="process.py",
    scriptArg="iterations=100,tolerance=0.001"
)
```

#### 完整示例
```python
import time

# 获取控制器
controller = getFlowSheetController()

# 启动计算任务
process_id = controller.runComputeTask(
    "complex_optimization",
    scriptFile="advanced_script.py",
    timeOutSeconds=1800,  # 30分钟
    scriptArg="maxIter=500,tol=1e-6,output=result.csv"
)

if process_id > 0:
    print(f"任务已启动，进程ID: {process_id}")
    
    # 获取日志文件
    log_file = controller.getProcessLogFile(process_id, "complex_optimization")
    print(f"日志文件: {log_file}")
    
    # 监控进程状态
    while controller.isProcessRunning(process_id):
        print("任务运行中...")
        time.sleep(5)
    
    print("任务完成！")
else:
    print("启动任务失败")
```

#### 批量运行任务
```python
tasks = [
    {"name": "task1", "script": "script1.py", "timeout": 300, "args": "mode=fast"},
    {"name": "task2", "script": "script2.py", "timeout": 600, "args": "mode=normal"},
    {"name": "task3", "script": "script3.py", "timeout": 900, "args": "mode=precise"}
]

process_ids = []
for task in tasks:
    pid = controller.runComputeTask(
        task["name"],
        scriptFile=task["script"],
        timeOutSeconds=task["timeout"],
        scriptArg=task["args"]
    )
    if pid > 0:
        process_ids.append(pid)
        print(f"启动任务 {task['name']}，进程ID: {pid}")

print(f"已启动 {len(process_ids)} 个任务")
```

### 相关API函数

| 函数 | 说明 | 返回值 |
|------|------|--------|
| `runComputeTask(...)` | 启动计算任务 | 进程ID（>0表示成功，-1表示失败） |
| `isProcessRunning(processId)` | 检查进程是否运行中 | True/False |
| `getProcessLogFile(processId, taskName)` | 获取进程日志文件路径 | 日志文件路径（字符串） |

## 调试技巧

### 测试命令行参数
在开发和调试阶段，可以先用较短的超时时间测试：

```bash
xRto.exe --computeTask test --projectFile test.xrto --timeOut 10
```

### 查看脚本参数传递
在脚本开始处打印参数，确认传递正确：

```python
import os
print(f"XRTO_SCRIPT_ARG = {os.environ.get('XRTO_SCRIPT_ARG', 'NOT SET')}")
```

### 日志输出
计算模式会输出详细的日志信息到控制台：
- 启动信息
- 工程文件加载状态
- 脚本执行状态
- 超时设置
- 执行结果

可以重定向输出到文件：
```bash
xRto.exe --computeTask myTask --projectFile project.xrto > output.log 2>&1
```
